## Datasets

Download from Kaggle and place each CSV in the `datasets/` folder:

1. **Telco Customer Churn** (classification)
   https://www.kaggle.com/datasets/blastchar/telco-customer-churn
2. **Mall Customer Segmentation** (clustering)
   https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python
3. **Groceries Dataset** (association rules)
   https://www.kaggle.com/datasets/heeraldedhia/groceries-dataset

## Setup

1. **Create a virtual environment**
```bash
   python -m venv venv
```
2. **Activate it**
   - macOS / Linux: `source venv/bin/activate`
   - Windows: `venv\Scripts\activate`
3. **Install dependencies**
```bash
   pip install -r requirements.txt
```

> **macOS note:** XGBoost and LightGBM require OpenMP.
> If you hit a `libomp` error, run: `brew install libomp`

## Running the project

### 1. Run the notebooks first
The Streamlit app loads trained models from `models/` and results from `results/`.
These are produced by the notebooks, so run them top-to-bottom **once**:

```bash
jupyter notebook
```

Then open and run:
- `classification.ipynb`
- `clustering.ipynb`
- `association.ipynb`

This populates `models/` and `results/` with the artifacts the UI needs.

### 2. Launch the UI

```bash
streamlit run streamlit_app.py
```

The app opens in your browser at `http://localhost:8501`.
